# **App Name**: QuizMaster Pro

## Core Features:

- User Authentication: Secure user sign-up and login functionality using Firebase Authentication.
- Quiz Configuration: Allow students to select the number of MCQs they want in a quiz session.
- Question Fetching and Display: Fetch random questions and their corresponding MCQs from the Excel file and display them one by one.
- Answer Submission and Validation: Allow students to select an answer option and submit it for validation.
- Real-time Feedback: Display instant feedback on whether the answer is correct or incorrect.
- Score Calculation and Display: Calculate the final score based on the number of correct answers and display it at the end of the quiz.
- Dynamic Difficulty Adjustment: Use AI to estimate the test taker's aptitude; adjust difficulty during the test in response. The AI will act as a tool to maintain engagement during the quiz.
- Excel Data Integration: Integrate with the provided Excel file to fetch and update questions, MCQs, and correct answers.

## Style Guidelines:

- Primary color: Deep sky blue (#3498db), reminiscent of a clear digital sky. This color inspires focus, learning, and clarity.
- Background color: Very light blue (#ecf0f1), which will provide a clean, distraction-free background conducive to concentration.
- Accent color: Sea green (#2ecc71), used to highlight correct answers and important UI elements; sea green implies growth and progress.
- Font: 'Inter' (sans-serif) for both headlines and body text to ensure optimal readability and a clean, modern user interface.
- Use clean, minimalist icons to represent different quiz categories and actions, ensuring clarity and ease of use.
- Employ a clean and intuitive layout that adapts to different screen sizes, ensuring a consistent user experience across devices.
- Incorporate subtle animations for transitions and feedback to enhance user engagement without being distracting.
